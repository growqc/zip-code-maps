The file county-zip-mapper.py is a Python app that takes a set of counties and returns all of the zip codes that cover at least a portion of those counties. Cities are the postal city name. County is the county where the postal city is located, even if that was not one of the given set of counties. Zip codes that have no Census shapefile polygon have the note "PO Box only." Map polygons are colored by the target county with the largest geographic overlap with the Zip code region.
Output files are .html and .csv. QC_zip_code and QC_zip_codes_by_county.csv are the output for the 6-county Quad Cities Combined Statistical Area.
